### Prerequisites

1. Clone or download this repository
2. Open `index.html` in your web browser

### Local Development

For development with live reload, you can use:

```bash
# Using Python
python -m http.server 8000

# Using Node.js with http-server
npx http-server

# Using PHP
php -S localhost:8000
```

Then open `http://localhost:8000` in your browser.

## ğŸ”§ Customization

### Update Personal Information

1. Edit `index.html` to update:
   - Name and titles
   - Contact information
   - Work experience
   - Education details
   - Skills and expertise

2. Replace the PDF resume:
   - Add your resume PDF to `PDF resume/` folder
   - Update the link in the "Download Resume" button

### Modify Colors

Edit the CSS variables in `styles.css`:

```css
:root {
    --primary-color: #2563eb;
    --secondary-color: #10b981;
    --accent-color: #f59e0b;
    /* ... other variables */
}
```

### Add Profile Image

Replace the placeholder in the hero section:

```html
<div class="profile-img-wrapper">
    <img src="path/to/your/image.jpg" alt="Gilbert Baidya">
</div>
```
